<ExpandableSection title="AI Summary">
This talk explores the rapidly expanding attack surface of AI agents -- autonomous systems that use LLMs to interact with tools, code, and external services. Starting with Agent 101 fundamentals (what they are, how coding and research agents work), the presentation moves into enumeration techniques: identifying whether a target is a chatbot or a full agent, mapping its inputs and outputs, fingerprinting the underlying model, and probing its modalities. The attack section walks through the OWASP LLM Top 10, then demonstrates real-world exploits: ANSI escape code injection in CLI agents (turning plain text into a weapon), prompt injection attacks against GitHub Copilot, jailbreak techniques (Marinara presets, abliterated models), and output-based attacks (markdown exfiltration, XSS, CSV/LaTeX injection). The second half focuses on supply-chain threats: malicious MCP (Model Context Protocol) servers that poison tool descriptions, exfiltrate data, and execute living-off-the-land attacks; malicious Skills that harvest credentials, execute remote code, and inject shadow context into agent sessions. The talk concludes with the emerging frontier of Agent-to-Agent (A2A) attacks -- where AI agents target other AI agents -- referencing the hackerbot-claw incident and discussing defensive architectures for building secure AI harnesses.
</ExpandableSection>

## Topics Covered

- **Agents 101** -- What AI agents are, how coding and research agents work
- **Enumeration** -- Feature identification, I/O mapping, LLM fingerprinting
- **Attacks** -- ANSI injection, prompt injection (CoPilot), jailbreaks, output attacks
- **Malicious MCPs** -- Tool poisoning, data exfiltration, LOTL attacks
- **Malicious Skills** -- Credential theft, remote code execution, shadow context
- **A2A Attacks** -- Agent-to-agent threats, the hackerbot-claw incident, defensive architectures
