Hugo dead simple theme

Apr 1, 2020    #hugo  
  • Installation

Simple hugo theme to be dead easy on the reader.

Installation

  • Initialize go module

    hugo mod init <module_path>

  • Add theme to your config.yml

    module:
  imports:
    - path: github.com/barklan/hugo-dead-simple

  • Fetch theme

    hugo mod get -u
